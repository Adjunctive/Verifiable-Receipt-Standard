VRS Contracts

Canonical contract ABIs and an implementation-agnostic conformance suite for the VRS Ethereum State Profile. ABIs and tests only — not an implementation.

Interface selectors are verified to match the reference implementation, and the conformance suite has been validated 13/13 against it. See the README and Specification Appendix B.

Interfaces (Solidity — the ABI)

  • VRSTypes.sol — structs & enums (normative field order/types)
  • IEntitlementRegistry.sol — ownership / status / lifecycle
  • IAttestationRegistry.sol — issuer identity
  • IAnchorRegistry.sol — proven time
  • IReceiptVerifier.sol — stateless proof verifier
  • IMintAllowanceRegistry.sol — batches & serial nullifiers
  • INoticeRegistry.sol — recall notices (draft)
